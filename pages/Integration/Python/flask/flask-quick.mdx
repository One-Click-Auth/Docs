# Quick Start with AI using TrustAuthX 🚀

## Set Up and Activate Virtual Environment (Highly Recommended) 🛠️✅

```bash copy showLineNumbers filename="bash"
py -m venv . 
```

<Steps>
## Step 1: Install TrustAuthX 📦

```bash copy showLineNumbers filename="bash"
pip install trustauthx
```

## Step 2: Login to TrustAuthX 🔑

```bash copy showLineNumbers filename="bash"
trustauthx login me -k <API_KEY> -s <SECRET_KEY> -o <ORG_ID>
```

## Step 3: Generate a flask Project Structure 🏗️

```bash copy showLineNumbers filename="bash"
trustauthx neuroform flask
```

## Step 4: Start the flask Project 🎉

```bash copy showLineNumbers filename="bash"
trustauthx start flask
```
</Steps>
