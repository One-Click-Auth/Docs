# Quick Start with AI using TrustAuthX 🚀

<Steps>

## Step 1: Install TrustAuthX 📦

```bash copy showLineNumbers filename="bash"
pip install trustauthx
```

## Step 2: Login to TrustAuthX 🔑

```bash copy showLineNumbers filename="bash"
trustauthx login me -k <API_KEY> -s <SECRET_KEY> -o <ORG_ID>
```

## Step 3: Generate a Framework  Project Structure 🏗️

```bash copy showLineNumbers filename="bash"
trustauthx neuroform Framework 
```

## Step 4: Generate a virtual Environment and Activate it ( **Strongly Recommend ) ✅

```bash copy showLineNumbers filename="bash"
python -m venv .
```

## Step 5: Start the Framework  Project 🎉

```bash copy showLineNumbers filename="bash"
trustauthx start Framework 
```
</Steps>
